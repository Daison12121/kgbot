    🎯 ПРОЦЕСС ДЕПЛОЯ (КРАТКО)
═══════════════════════════════════════════════════════════════════

1. Создайте сервер на Vultr
   → Ubuntu 22.04, $5/месяц, Frankfurt

2. Загрузите код на GitHub
   → Создайте приватный репозиторий
   → git push

3. Подключитесь к серверу
   → ssh root@YOUR_SERVER_IP

4. Установите бота
   → git clone
   → ./deploy.sh

5. Настройте .env
   → nano .env
   → Вставьте данные

6. Запустите бота
   → sudo systemctl start telegram-bot

7. Готово! 🎉

Подключение к серверу:
   ssh root@YOUR_SERVER_IP

Статус бота:
   sudo systemctl status telegram-bot

Логи в реальном времени:
   sudo journalctl -u telegram-bot -f

Перезапуск бота:
   sudo systemctl restart telegram-bot

Обновление бота:
   cd ~/telegram-bot && ./update.sh

Бэкап данных:
   cp ~/telegram-bot/data.json ~/telegram-bot/data.json.backup.$(date +%Y%m%d)

    🆘 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК
═══════════════════════════════════════════════════════════════════

Бот не запускается:
   sudo journalctl -u telegram-bot -n 50

Бот не отвечает:
   sudo systemctl restart telegram-bot

Ошибка конфигурации:
   cd ~/telegram-bot
   source venv/bin/activate
   python check_config.py

Забыли пароль от сервера:
   Vultr → Server → Settings → Reset Root Password
